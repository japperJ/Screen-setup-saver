Architecture Overview

Screen Setup Saver — Python tray app with settings window

Core Modules
main.py Entry point — wires everything together, starts tray + hotkey listener
tray.py System tray icon — right-click menu for quick save/restore/open settings
capture.py Walks all visible windows via win32gui — records exe path, title, position, state
browser.py Connects to Chrome/Edge debug port (CDP) — fetches all open tabs per window
restore.py Reopens apps via subprocess, repositions windows, reopens browser tabs
profiles.py Load/save JSON profiles to %APPDATA%\ScreenSetupSaver\profiles\
hotkeys.py Registers global hotkeys — save, restore last-used, open settings
settings_ui.py tkinter window — list profiles, rename, delete, edit hotkeys, run setup wizard

Dependencies
pywin32 pystray keyboard websocket-client Pillow tkinter (built-in)
CDP one-time setup: The app auto-creates desktop shortcuts for Chrome and Edge with --remote-debugging-port flags. First-time wizard walks you through it. After that, launching Chrome/Edge via these shortcuts enables URL capture.
