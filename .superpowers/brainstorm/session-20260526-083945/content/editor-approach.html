How should the JSON editor look?

You have a saved profile — pick how you'd like to edit its raw JSON

A

Inline editor in the app window

A text area opens in the Settings window (right panel) showing the raw JSON. You edit it directly, then click Save. The profile list stays visible on the left.

Popup dialog window

Clicking "Edit JSON" opens a separate floating window with the JSON editor. Larger editing area. Closes on Save or Cancel.

C

Open in system text editor

Clicking "Edit JSON" opens the .json file directly in Notepad (or your default .json handler). You save there. The app reloads the profile when you return.
